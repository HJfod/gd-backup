For the app to work, you need to set the path of your CCLocalLevels file.

Most likely it will have been set for you already, but if it fails to load, locate your CCLocalLevels file and try again.

To set the path, click set. Browse let's you browse for the path.

The path has to lead to your CCLocalLevels.dat file.


Backups are the same as saving your data in GD, except that the files are saved locally and you can have multiple at the same time.

To create a backup, click Make a new backup. This will create a backup of your current game files.

If you have made backups before, you can add them by clicking Import. External backups can be located anywhere on your computer.

To load a backup, hover over the backup option and click Switch. The app will ask you if you'd like to save your current GD progress. Press Save stats to make a new backup out of your current GD data, or Discard to replace your current GD data with the backup.

Click Refresh to reload your backup folders.

The ability to view info about backups, such as user stats and levels, will be added later.


To import a level, click to select it or drop the file onto the import file area.

Clicking import puts the level at the top of your GD levels.

Click Analyze to view info about the level such as creator, object count, etc.


To export a level, select it from the list. You can use search to find it.

By default, levels are exported to the levels folder located in the app's files. You can find the folder by scrolling down and clicking "Open levels folder".

If you'd like to export to some other location, uncheck Export to default location and then browse to select your export location.

You can also export multiple levels by just selecting multiple ones.

When hovering a level in the list, you can see a button that says details. Click it to view info about the level such as creator, object count, etc.
